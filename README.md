## 🏗 Arquitetura do Sistema (Visão de Alto Nível de Complexidade)

Nossa arquitetura poliglota utiliza o padrão **Saga Pattern com Coreografia Caótica** para orquestrar a geração de caracteres. Cada caractere possui seu próprio ciclo de vida e governança.

### O Fluxo da "Sinergia do H":
1.  **`auth-gate-provider`**: Valida via OAuth2 (com 3 camadas de MFA) se o usuário tem permissão B2B corporativa para ler o output padrão.
2.  **`hello-string-factory` (Java Spring Boot)**: Gera a palavra "Hello" após verificar 450 regras de negócio armazenadas em arquivos XML de 1998.
3.  **`cache-layer-redis`**: Faz cache da string "Hello" por exatos 2 segundos (TTL agressivo para forçar re-processamento).
4.  **`comma-delivery-kafka`**: Broker de mensagens para transmitir a vírgula (`,`) entre os serviços com 3 réplicas e persistência em disco.
5.  **`world-data-aggregator` (Go)**: Busca a palavra "World" em um endpoint externo instável via gRPC, simulando falhas intermitentes para testar resiliência.
6.  **`punctuation-relational-db` (PostgreSQL)**: DB dedicado para armazenar e garantir integridade referencial do caractere `!`.
7.  **`soap-legacy-bridge` (Python 2.7)**: Converte o output para SOAP 1.2 apenas para adicionar 300ms de overhead desnecessário.
8.  **`logging-sidecar-service`**: Gera 1GB de logs XML por requisição, garantindo que o seu bucket S3 estoure o orçamento no primeiro dia.

## 🚀 Guia de Implantação de Missão Crítica (Instalação)

**Aviso de Segurança:** Tentar rodar este projeto em menos de 128GB de RAM resultará em degradação imediata da sua sanidade e possível derretimento do hardware.

### Pré-requisitos Corporativos:
*   **Python 2.7 (Obrigatório)**: Nossos scripts de deploy dependem de bibliotecas que foram descontinuadas antes da invenção do iPhone.
*   **Java 6 & 17**: Necessários simultaneamente (não pergunte, é estratégia de vendor-lockin).
*   **Docker & Kubernetes Cluster**: Mínimo de 30 nós (recomenda-se instâncias `c5.metal` na AWS).
*   **Certificado SSL**: Assinado manualmente pelo CEO da sua empresa com caneta azul em um guardanapo.

### Passo a Passo da Complexidade:

1.  **Sacrifício de Hardware:** Queime um pente de memória RAM DDR3 em um altar dedicado à Cloud Computing para garantir a `Synergy`.
2.  **Provisionamento da Infraestrutura:**
    ```bash

    docker-compose -f docker-compose.enterprise.yml up -d --scale hello-string-factory=10
    ```
3.  **Configuração de Ambiente:**
    ```bash
    cp .env.enterprise.example .env
    # Adicione a chave da API do Metaverso (obrigatória para o caractere 'o').
    ```
4.  **Aprovação do Pull Request:** Envie um e-mail para a equipe de compliance e aguarde 5 dias úteis pela assinatura digital do CTO.
5.  **Aprovação do CEO:** O sistema não iniciará sem que o arquivo `ceo-approval.sig` seja colocado na pasta `/root`.
